# Sorting-Hat

A CLI app that allows user to take a "Sorting Hat" quiz to be placed in one of Harry Potter Hogwarts Schools.

## Play the quiz


```
ruby bin/run.rb

```

You can use keyboard arrows (up & down) and press ENTER key for selecting choices.
